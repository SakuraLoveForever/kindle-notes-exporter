## 功能介绍：
1. 支持点击或拖拽上传 My Clippings.txt
2. 支持单本书籍笔记导出，也支持所有书籍统一导出

## 使用方法：
1. 先将Kindle连接电脑，在Kindle磁盘中找到document文件夹，找到My Clippings.txt
2. 打开工具网址，点击或拖拽My Clippings.txt
3. 单本笔记或所有笔记统一导出
